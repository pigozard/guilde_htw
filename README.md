# Guilde HTW - Application de gestion de guilde WoW

## Description
Application Rails pour gérer la guilde "Highway to Wipe" sur World of Warcraft.

## Fonctionnalités
- Authentification des membres
- Calendrier des événements/raids
- Gestion des participations

## Installation
1. `bundle install`
2. `rails db:create db:migrate`
3. `rails server`

## Stack technique
- Ruby 3.3.5 / Rails 7.1.6
- PostgreSQL
- Bootstrap 5
